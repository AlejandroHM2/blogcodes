# usar el custom authentication de Azure Mobile Services .NET back-end

Para más información consulte el [artículo relacionado](http://nicolocodev.wordpress.com/2014/12/19/como-usar-el-custom-authentication-de-azure-mobile-services-net-back-end/)


## Uso

Para lanzar la aplicación cliente ejecute el acceso directo src/client/server/launch-iisexpress. Este inicia el iis express en el puerto 8000.
Es necesario estar conectado a internet para poder ejecutar la aplicación cliente. Esta hace uso de CDNs para usar jQuery y React.js
