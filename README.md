Ejemplos de código de mi blog
=========



En algunas de las entradas de mi blog, sobre todo en las de los primeros años, agrego el proyecto de ejemplo para descargar. Con el objetivo que el lector pueda interactuar directamente con código que funciona y no tenga que copiar y pegar las n porciones de código que componen el artículo.



Todo el código que publiqué se encontraba en una carpeta pública de mí SkyDrive (hoy OneDrive), producto que ha tenido una serie de actualizaciones y, que por extrañas razones, ha extraviado algunos de los enlaces que en mi blog se encontraban.



Aprovechando que no son muchas las entradas que al día de hoy tenían ejemplos de código, he decidido moverlas a este espacio, no solo con el objetivo de actualizar las URLs, sino para aprovechar que mucha gente tiene una cuenta en GitHub y puede sugerir de forma directa cambios y/o mejoras al código. El que ha sido uno de mis objetivos además, la retroalimentación continúa.



Aprovecharé este repositorio para subir, además de los ejemplos de mi blog, otras pruebas de código y concepto, que por pequeñas o faltas de fundamento teórico o de investigación, no ameritan la escritura de un artículo.
